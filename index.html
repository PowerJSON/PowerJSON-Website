PowerJSON

—— json's improved data format.



It supports comments、multi-line string、import file、operators and more feature that improved from json.

// comment
/*
    comment
 */

{
    text: 'hello world',
    MultiLineText: `
        Welcome
        to
        PowerJSON!
    `,

    // file and import
    myFile: new File('./file.txt'),
    importPJSONFile: new PJSON('./index2.pjson'),
    getFile: new GET('http://www.powerjson.org'),

    // operator
    tenDaySeconds: 60 * 60 * 24 * 10
}
